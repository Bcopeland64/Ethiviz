# EthiViz - Streamlit Interface Guide

This document provides instructions for using the original Streamlit-based interface of the EthiViz platform. For the primary React frontend and Flask API, please refer to the main `README.md`.

## Overview

The Streamlit interface provides an interactive way to analyze text and image data for cultural bias through multiple ethical traditions.

## Features (Streamlit Mode)

- **Streamlit UI**: Modern, sleek dark-themed user interface for intuitive analysis.
- Interactive dashboards and visualizations.

## Installation (Streamlit Mode)

### Using the Setup Scripts (Recommended)

These scripts are designed to set up the environment primarily for the Streamlit application.

#### On Linux/macOS:
```bash
# Clone the repository (if you haven't already)
# git clone https://github.com/yourusername/ethiviz.git
# cd ethiviz

# Make the setup script executable and run it
chmod +x setup.sh
./setup.sh

# Activate the virtual environment
source activate-ethiviz.sh

# Run the application
streamlit run Scripts/app.py 
# Note: The original README mentioned 'streamlit run app.py'. 
# Assuming app.py is now in Scripts/
```

#### On Windows:
```bash
# Clone the repository (if you haven't already)
# git clone https://github.com/yourusername/ethiviz.git
# cd ethiviz

# Run the setup script
setup.bat

# Activate the virtual environment
activate-ethiviz.bat

# Run the application
streamlit run Scripts/app.py
```

### Manual Installation (Streamlit Mode)

If you prefer to set up manually for the Streamlit application:

```bash
# Clone the repository (if you haven't already)
# git clone https://github.com/yourusername/ethiviz.git
# cd ethiviz

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate.bat

# Install dependencies from the main requirements file
pip install -r requirements.txt 
# Ensure Streamlit and other visualization libraries are included

# Download spaCy model (required for text analysis)
python -m spacy download en_core_web_md # Or en_core_web_sm if preferred for lighter use
```

## Usage (Streamlit Mode)

The easiest way to use EthiViz's original UI is through its interactive Streamlit interface:

```bash
# Ensure you are in the root directory of the project
# Activate your Python virtual environment if you created one
# e.g., source venv/bin/activate or venv\Scripts\activate.bat

# Launch the Streamlit app 
python Scripts/run.py 
# Or directly with streamlit:
# streamlit run Scripts/app.py
```

This opens a web interface (usually on port 8501) where you can:
- Upload text data (CSV, Excel) and images
- Use included sample text data
- Use included sample images
- Select ethical traditions for analysis
- Configure advanced settings
- Run analyses and view interactive results
- Save results.

## Command Line Interface (via `run.py`)

The `Scripts/run.py` script, originally designed with the Streamlit application in mind, can also be used for batch processing or integration with other tools:

```bash
# Ensure your Python environment is activated.

# Analyze text data
python Scripts/run.py --text-data path/to/textdata.csv

# Analyze images
python Scripts/run.py --image-dir path/to/image/directory

# Analyze both text and images
python Scripts/run.py --text-data path/to/textdata.csv --image-dir path/to/image/directory

# Specify ethical traditions to use
python Scripts.run.py --text-data path/to/textdata.csv --traditions western,ubuntu,confucian,islamic

# Set image analysis feature extraction level
python Scripts/run.py --image-dir path/to/image/directory --feature-level medium

# Specify output directory and dashboard port (if applicable for run.py's dashboarding)
python Scripts/run.py --text-data path/to/textdata.csv --output-dir ./my_results --port 8080 
# Note: The --port argument might be specific to a dashboard generated by run.py, not the Streamlit app's port.
```
